# Credit Card Transaction & Customer Analysis

## Overview
Interactive Power BI project analyzing credit card transaction and customer behavior data using SQL, Excel, DAX, and Power BI.

## Tools Used
- Power BI
- SQL
- Excel
- DAX formulas

## Dashboards Included
1. **Transaction Dashboard**
   - Total revenue & transaction count
   - Revenue by card category (Blue, Silver, Gold, Platinum)
   - Weekly revenue trends
   - Transaction modes (Chip, Swipe, Online)

2. **Customer Dashboard**
   - Customer segmentation by job type
   - Spending behavior by customer group
   - Revenue contribution by customer demographics

## Output
- Interactive dashboard with filters and KPIs for business insights
